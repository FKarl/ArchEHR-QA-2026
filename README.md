# ArchEHR-QA-2026

This repository contains the code for our accepted paper for the ArchEHR-QA subtask at the CL4Health workshop @ LREC 2026.

## Context

ArchEHR-QA ("Archer") focuses on question answering over electronic health records (EHRs) in a clinical setting.

## Notes

- This repository accompanies the workshop submission.
- Further details can be found in the paper: [https://arxiv.org/abs/2603.13962](https://arxiv.org/abs/2603.13962)
